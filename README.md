# 🧾 BiyaLog — Vintage Receipt Web

BiyaLog adalah web sederhana untuk mencatat lagu dalam bentuk struk aesthetic vintage.

## ✨ Features
- Input lagu manual
- Tampilan receipt vintage
- UI aesthetic (Tailwind CSS)
- Efek sobekan (torn paper)

## 🛠️ Tech Stack
- PHP
- MySQL
- Tailwind CSS

## 📸 Preview
Vintage receipt style inspired by Spotify Receiptify

## 🚀 Cara Menjalankan
1. Jalankan XAMPP / Laragon
2. Import database
3. Jalankan di:
http://localhost/biyalog
